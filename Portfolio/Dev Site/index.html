Feed-A-Star-Mole Game

Feed-a-mole is a more humane approach to the classic "Whack-a-mole" game. In this game, players click on hungry star moles to feed them worms. When enough worms are fed, the winner is rewarded with the image of a cute star mole contentedly napping. JavaScript using animation frames make it function. Link to the page here.

  • HTML
  • CSS
  • JavaScript
